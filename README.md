# FinSight AI v2

AI-Powered Financial Analysis Platform with a Neobrutalist styling.

## 🚀 Cloud Deployment

You can deploy the backend of this application to Render with a single click:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/deadiop/FinSight-AI)

### Setup Instructions

1. Click the **Deploy to Render** button above.
2. In the Render Dashboard, provide the following required environment variables:
   - `DATABASE_URL`: Your Neon PostgreSQL connection string (pooler link).
   - `GROQ_API_KEY`: Your Groq API key (`gsk_...`).
3. Click **Apply** or **Deploy**.
4. Once deployed, copy your Render web service URL (e.g., `https://finsight-backend-xxxx.onrender.com`).
5. Open your live frontend app: **[https://xpense-3e99e9.netlify.app](https://xpense-3e99e9.netlify.app)**
6. Click the settings gear icon in the top right, paste the Render URL, and save!
